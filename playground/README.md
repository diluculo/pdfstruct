# Playground

A local sandbox for exercising the CLI against real PDFs.

Drop your own input documents (and any output files you want to inspect) here. Everything in this directory is ignored by git — keeping private, copyrighted, or large PDFs out of the repository — except this `README.md` and the local `.gitignore` itself.

For PDFs that ship with the repo and are safe to redistribute, see [`tests/fixtures/`](../tests/fixtures/) instead.
